# Feature Dictionary

K-ETS market VWAP 타겟 모델링에 사용하는 wide feature 명세이다.

## 1. 컬럼 기준

| 컬럼 | 설명 |
|---|---|
| `feature_name` | 모델 입력 컬럼명 |
| `feature_label_ko` | 그래프와 표에 표시할 한글 라벨 |
| `feature_description_ko` | 변수 정의와 계산 기준 |
| `feature_group` | 모델링용 feature group |
| `source_table` | 원천 테이블 또는 processed artifact |
| `join_rule` | 기준일 결합 규칙 |
| `baseline_candidate` | 05 baseline 기본 후보 포함 여부 |

## 2. Feature group 정의

| feature_group | 한글 라벨 | 설명 |
|---|---|---|
| `market_internal` | 시장 내부 | market_vwap, 거래량, 빈티지 구조, lag/rolling 변수 |
| `power_epsis` | 전력/EPSIS | 전력수급, SMP, 발전연료비, 전력거래량 |
| `weather_kma` | 기상/KMA | 기온, 강수, 습도, 풍속, HDD/CDD |
| `external_market` | 외부시장 | 국제유가, EUA, LNG/석탄/천연가스 가격 |
| `auction` | KRX 경매 | 배출권 경매 가격, 수량, 참여자, 체결률 |
